## **Nextcloud Documentation**
1. Hardware & Materials
2. Server: Old Asus Vivobook S15 (1.8TB storage)
3. Installation Media: USB flash drive
4. Operating System: Ubuntu Server 24.04.2 LTS
5. Tools: Balena Etcher, Claude AI

-- 
## **Goal:** Replace Google Photos and Google Drive with a self-hosted, private alternative using Nextcloud on a repurposed laptop. This homelab will provide secure, permanent storage for personal files and photos with full control over data.
-- 

## Project Scope
> Phase 1: Install Ubuntu Server on the Vivobook
> Phase 2: Install and configure Nextcloud
> Phase 3: Set up local network access
> Phase 4: Configure internet access (remote access with security)
>Storage Plan: ~300GB for Nextcloud, 1TB+ for Jellyfin (media server), with room to spare
